About Me

My Name is Jackson Levitt. I am a student at Rochester Institute of Technology studying Game Design and Development and immersing myself in Economics. Since I was little, I always enjoyed solving problems, from puzzles to games to code. I love the challenges of coding and love when it all comes together.

C# Class Library Framework

Trainworks

Modding Framework for Monster Train

Tools

  • C#,
  • BepInEx,
  • HarmonyLib

Trainworks Modding Tools is a C# Class Library used in combination with BepInEx and HarmonyLib to allow for patching the Rogue-like Card Game, Monster Train. It was created by me and some other modders to allow for modding on Monster Train. I learned alot about CIL, AssetBundles, Asset Loading, and Patching. I personally implemented alot of the plugin management code.

Javascript Web App

Everything Sealed

Magic the Gathering Card Pool Generator

Tools

  • HTML,
  • CSS,
  • Javascript

Everything Sealed Card Pool Generator is a Javascript Web App developed to allow for the generation of a collection of Magic the Gathering cards used to play "Everything Sealed". It was created by me for usage by a group of friends. I learned about Web APIs and Queries. I overcame issues with bad requests, wherein improperly data was returned from the API.

Javascript Web App

Ibex Idle

Idle Game

Tools

  • HTML,
  • CSS,
  • Javascript,
  • Pixi.JS

Ibex Idle is an Idle game, in which you click, buy upgrades, and chill out. It was created by me as I wished to understand the implementation of this rather simple genre. I got to experiment with PIXI.JS and learn about how to create simple Javascript games. I enjoyed streamlining UI and developing a formula for the Ibex's horns.
